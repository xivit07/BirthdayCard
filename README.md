# Friends Birthday Website

## Open in VS Code
1. Extract the zip.
2. Open the folder in VS Code.
3. Run:
   npm install
   npm run dev
4. Open the local URL shown by Vite.

## Notes
- The birthday video is already included at `public/videos/birthday-memory.mp4`.
- Main editable file: `src/FriendsBirthdayWebsite.tsx`
- Change the friend details at the top of that file:
  - `FRIEND_INITIALS`
  - `FRIEND_NAME`
